배송정보 연계 신청안내입니다.

1. Open-API 신청 방법

- 신청자 이름, 사용할 서비스의 형태소개 등에 대해서 이메일을 작성하여sweetracker.kr@gmail.com 으로 등록키 신청을 해주십시요.

- 또는, 아래의 Open-API 신청 서류를 작성하여 주십시요.

2. 스윗트래커는 7일~ 15일 이내에 신청되어진 내용을 검토한후, 문제가 없을시 등록된 이메일을 통하여 필요한 동작 Key를 발급합니다.

3. 개발자는 발급된 Key를 가지고 연계 서비스를 API 문서 기준으로 사용하시면 됩니다.

4. 무료서비스의 경우에는 하루 1천건, 월 1만건 이하로 제한되어지며, 해당 제약 조건은 스윗트래커의 서비스 상황에 따라 공지없이 바뀔 수 있습니다.

5. 무료서비스의 경우에는 사전에 공지없이 시스템이 중단되거나, 사용방식의 변경이 있을 수 있습니다.

6. 무료서비스를 사용하는 경우에는 다음과 같이 '출처'를 명시하셔야 합니다.

- 본 정보는 스마트택배에서 제공받는 정보로, 실제 배송상황과 다를 수 있습니다.

- 사용하는 서비스의 배송조회 페이지에 위의 문구가 표시되어야 합니다.

7. 해당 사용조건이 만족하지 않는 경우 무료서비스가 사전동의없이 중단될 수 있습니다.
