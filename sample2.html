아래 샘플은 배송정보 API를 이용해 소규모 쇼핑몰이나 블로그 기반의 쇼핑몰에서도 유용하게 사용이 가능하도록 한 샘플입니다.

배송정보 API를 이용해 쇼핑몰의 관리 페이지나 관리 프로그램에서 쇼핑몰에서 판매한 물품이 어떻게 이동하고 있는지 확인할 수 있습니다.

연동된 배송정보는 쇼핑몰의 관리페이지에서 필요한 형태로 재배치가 가능합니다. 원하는 형태로 구성하십시오.

(주) 스윗트래커의 배송연계서비스를 기반으로 다음카카오에서 활용하고 있는 서비스를 예로 설명하겠습니다.

해당 서비스는 현재 다음 카카오의 모바일 기반 페이지에 제공되고 있으며, m.daum.net 으로 접속하셔서 '운송장번호'를입력하시면 검색되는 기능입니다.

CJ대한통운 687099730363 이라고 입력하고 검색하시면, 스윗트래커의 배송연계서비스를 통해서 배송정보를 검색하는 기능을 사용하실 수 있습니다.

아래의 화면은 스윗트래커에서 제공하는 배송연계API를 사용하여 실제 다음카카오의 모바일 웹에서 동작하는 기능 구현 예입니다.

다른, 택배사명과 운송장번호를 별도로 넣어서 검색 화면을 구동해 보시면 됩니다. ( 운송장 번호가 틀린 경우에는 배송정보가 리턴되지 않습니다. )

( 설명 : 제공: 스마트택배 <- (주) 스윗트래커의 안드로이드, 아이폰 앱 입니다.

이 기능과 동일하지만, 웹서비스 호출 횟수에 제한이 있는 버전이 Open-API로 제공됩니다.
